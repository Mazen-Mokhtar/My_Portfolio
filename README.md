# Mazen's Portfolio

## Project Overview
A modern, responsive portfolio website showcasing Mazen's skills, experience, and projects.

## Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- shadcn/ui

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Features
- Responsive design
- Smooth animations
- Interactive UI components
- Project showcase
- Contact form
